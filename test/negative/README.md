# Negative fixtures

Cases in this lane must fail closed with a stable diagnostic ID, essential
fields and source position, plus an assertion that no plausible generated
artifact survived. See [`docs/testing.md`](../../docs/testing.md).

Existing bootstrap, typed-C, HxcIR, body-lowering `HXC1001`, and post-body
`HXC1000` negatives remain mapped in their focused suite directories.
